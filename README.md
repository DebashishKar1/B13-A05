## Assignment 5

### Question

**Question 1:What is the difference between var, let, and const??**

Answer: var:function-scoped and re-declarable, let:block-scoped and re-assignable, const:block-scoped but cannot be re-assigned.

**Question 2: What is the spread operator (...)?**

Answer:Spread operator (...): It expands an array or object into individual elements/properties.

**Question 3: What is the difference between map(), filter(), and forEach()?**

Answer:map() returns a new array after transforming elements, filter() returns a new array with elements that pass a condition, and forEach() just loops through elements without returning a new array.

**Question 4: What is an arrow function?**
Answer:Arrow function: A shorter syntax for writing functions in JavaScript using =>.

**Question 5:What are template literals?**
Answer: Template literals: JavaScript strings written with backticks that allow embedded variables using ${}
